Landing Page



This Desktop Application serves as an Online Library of the Unic Location Karlsruhe.

Interacting with the Navigation Bar you can reach and use all implemented Functionalities.
It will adjust automatically depending on Application State and the Role of the currently logged in user (admin or user is possible).

Logged in Users with the Role user will be able to get an Overview over all the Books currently stored within the Library.
The Books can be sorted by Title, Author, Publisher and Page Count.

Logged in Users with the Role admin beyond that will be able to make Changes to the Library, including Adding/ Updating and Deleting Book Entries.
Adding/ Updating Books can be performed via Google Books Api or manually via Form Submission.
The Google Books Api allows to find many Books (not all) by ISBN so that you can simply type in any ISBN of your Choice in order to Add a Book to the Library or Update it in Case that a Book with this ISBN is already saved in the Library.

Hovering over Links and Images will show a Tool Tip providing information about the Linked Page.

If you click on the Logo in the Top Left Corner you will be redirected to this Landing Page.
